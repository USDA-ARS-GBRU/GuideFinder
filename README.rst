predictPAM: A python module to predict custom PAM sites in any small genome
==================================================================================================


# Usage
predictPAM -i [.gbkfile] -p [PAM motif] -o [outputfile]

# Example
predictPAM -i Pseudomonas_aeruginosa_PAO1_107.gbk -p "ATCG" -o test.txt --keeptemp
